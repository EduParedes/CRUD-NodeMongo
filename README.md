# Node CRUD MongoDB

This is a simple CRUD application using MongoDB.


# Step:
  1. Initialize your MongoDB service or Docker 
  
  ``` Windows ``` Command propmt
    
  ```
     > mongod
  ```

  ``` Docker```

  ```
   docker run --name mymongo -p 27017:27017 -d mongo
  ```
  
  2. Start CRUD app 
  
  ```
   npm run dev
  ```
